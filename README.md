# Swarm CLI Releases

This repository hosts public binary releases for the [Swarm](https://github.com/niclaslindstedt/swarm) CLI.

## Installation

Installation is simple:

```bash
curl -fsSL https://raw.githubusercontent.com/niclaslindstedt/swarm-cli-releases/main/install-swarm.sh | bash
```

## Supported Platforms

- Linux (x86_64, aarch64)
- macOS (x86_64, arm64)

## About Swarm

Swarm is a synchronous multi-agent workflow system for building large-scale applications with Claude Code. It prevents context overflow and incomplete implementations by breaking development into manageable iterations.

For more information, see the [Swarm](https://github.com/niclaslindstedt/swarm) project.
